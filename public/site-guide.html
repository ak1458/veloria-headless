🔥 Hot Sellers Section

The Hot Seller grid appears on the homepage at /#hot-sellers.

Where to Edit

WhatFile
Which bags appearsrc/config/hot-sellers.ts
Section heading & subtitlesrc/config/hot-sellers.ts
Label colorsrc/config/hot-sellers.ts

How to Change the Bags

  1. Go to WooCommerce → Products and find the bag you want.
  2. Copy the product ID (visible in the URL or the product list).
  3. Open src/config/hot-sellers.ts and replace one of the numbers in HOT_SELLER_IDS:
export const HOT_SELLER_IDS: number[] = [
  3459, // 🛍️ Freya
  3137, // 🛍️ The Amara
  3692, // 🛍️ The Vanya Sling
  3734, // 🛍️ Vivian
];

Keep exactly 4 entries for the grid to look correct.

How to Change Heading Text

In the same file, edit the HOT_SELLER_SECTION object:

export const HOT_SELLER_SECTION = {
  id: "hot-sellers",
  label: "Hot Seller",            // ← small label above heading
  title: "Your new heading here", // ← main heading text
  labelColor: "#b59a5c",          // ← label color (gold)
};

💕 Most Loved Styles

The tabbed product carousel on the homepage at /#most-loved-styles.

Where to Edit

WhatFile
Tab labels, categories, colors, product IDssrc/config/most-loved.ts
Section heading & subtitlesrc/config/most-loved.ts

How Each Tab Works

{
  slug: "tote-bag",
  label: "Tote",                             // Tab button text
  categorySlug: "tote-bag",                  // WooCommerce category slug
  allowedColors: ["black", "brown"],         // Only show these colors
  productIds: [3003, 3459],                  // Show these first (in order)
  maxItems: 8,                               // Max products in carousel
  viewAllHref: "/product-category/tote-bag", // "Explore" button link
},

To Add or Remove a Tab

  1. Open src/config/most-loved.ts.
  2. Add a new object to the tabs array or remove one.
  3. Each tab must have a unique slug.

To Change Heading

Edit eyebrow (small text above) and title in the config.

Trust Badges

The trust icons row below the accordion on every product page.

WhatFile
Badge list (title, subtitle, icon key)src/config/product-page.ts → PRODUCT_PAGE_TRUST_BADGES
SVG icon definitionssrc/components/ProductTrustBadges.tsx
Icon keys: leather, craft, secure, ethical, packaging, warranty, returns. To add a new icon, add an SVG to the BADGE_ICONS map in ProductTrustBadges.tsx.

Product Description Accordion

The accordion automatically parses the WooCommerce product description into sections using headings like "Size and Material", "Returns and Exchanges", "Warranty", and "Ethically Made".

No config file needed — just structure your WooCommerce description with those section headings.

🎟️ Scratch Coupon

The scratch card appears only on product pages. Users scratch to reveal a discount.

Where Things Live

WhatFile

Backend logic (weights, cookie, JWT)src/lib/scratch-coupon.ts
API endpoints (spin + status)src/app/api/lucky-draw/spin/route.ts
src/app/api/lucky-draw/status/route.ts

Discount Weights

// In src/lib/scratch-coupon.ts
{ discount: 5,  weight: 50 },   // 50% chance → 5% off
{ discount: 10, weight: 40 },   // 40% chance → 10% off
{ discount: 15, weight: 10 },   // 10% chance → 15% off

The visual reveal can show up to 50%, but the applied discount is always 5%, 10%, or 15%.

Each user can only scratch once per 30 days (tracked by cookie + IP).

💰 Coupons & Discounts

Automatic Tier Discounts

  • 1 item → 15% off (automatic)
  • 2+ items → 20% off (automatic)
  • Prepaid → Extra 5% off (automatic)

These are not configurable via config files. They are in src/lib/coupon-calculator.ts.

Manual Coupons (Influencer, Seasonal, etc.)

WhatFile
All manual coupon codessrc/config/coupons.ts

Follow the examples in the file to add new influencer or seasonal codes. All discounts are globally capped at 35%.

🚀 Deployment

  1. Make your edits to the config files listed above.
  2. Save the file(s).
  3. Push to Git: git add . && git commit -m "update config" && git push
  4. Vercel will auto-deploy within 1-2 minutes.
Tip: For simple text/color/product changes, you only need to edit the config files in src/config/. No component code changes needed.
